# Módulo — Painel analítico

**Versão do produto:** 7.0.2 · **Última revisão:** 2026-07-06

> **Índice de módulos:** [README.md](README.md) · **Rota:** `/dashboard/analytics`

Hub de **consultoria municipal**: cinco áreas (Resumo, Cadastro, Pedagógico, Censo, Finanças) sobre dados i-Educar, APIs públicas e caches locais.

---

## O que este módulo faz

| Capacidade | Descrição |
|------------|-----------|
| **Navegação por abas** | Ordem fixa Resumo → Finanças; impacto por aba documentado |
| **Métricas e Pulse** | Queries, indicadores e diagnóstico de performance |
| **Relatório PDF** | Exportação Serventec (ATM) por escopo municipal |
| **Integrações BI** | Estudo Power BI e exportações tabulares |
